O tema de hoje é machine-learning!

Introdução

Machine-learning é um aprendizado de máquina baseado em parâmetros usados para treinamento.

Uma forma simples de entender isso é lembrando do aprendizado humano. Nós aprendemos através do reforço de informação, isto é, quando nos é apresentada uma informação, nós criamos um conceito do que é aquilo e ele vai sendo moldado conforme recebemos mais informações.

Por exemplo:

Nós sabemos o que é uma flor, pois durante nossa criação fomos apresentados à várias categorias de flores.

Nós também sabemos o que é um cachorro, pois fomos apresentados a esse conceito durante nossa criação.

Uma vez que construímos o conceito de flor e cachorro, somos capazes de distinguir um do outro. Contudo, se não tivermos ideia do que é um barco e formos apresentados a isso, não saberemos classificá-lo

O machine learning também funciona da mesma forma. Criamos um modelo onde são apresentados dados para formação deste. Após este modelo ser treinado com as informações apresentadas, ele passa a ser capaz de prever um resultado buscado, baseando-se nos parâmetros de treinamento.

Um bom exemplo de aplicação disso é a previsão de preço de casas em uma dada cidade (esse é o exemplo tutorial do Kaggle, um excelente site de introdução à ciência de dados).

Referência desafio Kaggle de imóveis
    https://www.kaggle.com/c/home-data-for-ml-course

Para construir o modelo precisamos ter em mãos uma grande lista de residências composto com o maior número possível de informações (localização, área da residência, vagas de garagem, banheiros, quartos, etc). Além das informações desses imoveis, precisamos ter também acesso à informação que desejamos, no nosso caso, o preço de venda desses imóveis.

Assim, com as características desses imóveis e o preço de venda (dados históricos) é possível treinar um modelo que obtenha o preço de novos imóveis que venham a ser vendidos.

Entendido o funcionamento de machine learning, na aula de hoje faremos alguns exercícios sobre o tema.

Para começar, vamos tentar prever a nota da prova de Matemática baseando-se no resultado das outras provas. Para isso, vamos começar importando as bibliotecas padrão e os dados do estudo e separando os dados para a previsão:

Quando pensamos em funções matemáticas, lembramos da estrutura

    f(X) = y

Por conta disso é comum utilizar essa mesma notação em machine learning. Dessa forma:

    X = notas_entrada
    y = notas_saida

Um conceito importante em machine learning é a divisão dos dados em dois grupos:

  • dados de treinamento do modelo
  • dados de checagem do modelo

É muito importante fazer isso antes de iniciar a construção do modelo, pois isso evita que o resultado utilizado para treinamento do modelo seja o mesmo de checagem, gerando-se assim uma métrica errada de qualidade do modelo, isto é um modelo pode parecer muito bom para dados de treinamento, porém pode ser muito ruim para dados fora da amostra de treinamento.

Para isso o python possui uma biblioteca que facilita bastante a vida do usuário: sckit-learn.

    Referência para scikit-learn:
        https://scikit-learn.org/stable/

A expressão:

    seed=4321

(o número 4321 foi escolhido de forma aleatória)

Tem o objetivo de ser usado como uma "semente" que evita variação de resultados da quebra dos dados para simulação. Assim, devemos setar ramdom_state com o valor de seed para garantir que tenhamos uma reprodutibilidade do processo.
